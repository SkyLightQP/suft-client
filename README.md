## Branches
- `release`
    - 실제 서비스 코드
- `master`
    - 수프트 프론트엔드
- `develop`
    - 개발 코드

## 시작하기
### 의존성 설치
`yarn` (으)로 의존성 패키지를 다운로드합니다.

### 개발 모드 시작
`yarn start` (으)로 개발 모드를 시작합니다.

### 빌드
`yarn build` (으)로 빌드를 합니다.

### ESLint 검사
`yarn lint` (으)로 Lint 검사를 시작합니다.
